# ⚡ Быстрый старт (5 минут)

## Минимальные шаги для запуска:

### 1. Firebase (3 минуты)
1. https://console.firebase.google.com/ → Создать проект
2. Добавить Web-приложение → Скопировать конфигурацию
3. Realtime Database → Создать → Тестовый режим → Скопировать URL
4. Вставить в `firebase-config.js`

### 2. GitHub Pages (2 минуты)
1. Создать репозиторий на GitHub
2. Загрузить файлы:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/ВАШ_USERNAME/ВАШ_REPO.git
   git push -u origin main
   ```
3. Settings → Pages → Branch: main → Save

### 3. Готово!
Сайт доступен: `https://ВАШ_USERNAME.github.io/ВАШ_REPO/`

**Подробная инструкция:** см. `SETUP_GUIDE.md`
